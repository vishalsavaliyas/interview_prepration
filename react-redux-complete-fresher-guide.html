1. What is React?

React is a JavaScript library used to build user interfaces using reusable components.

2. Components

A component is a function that returns UI.

function Hello() {
  return <h1>Hello World</h1>;
}
Components are reusable building blocks of UI.

3. Props (Properties)

Props are data passed from parent to child. They are read-only. 

function Child({ name }) {
  return <h2>Hello {name}</h2>;
}

function Parent() {
  return <Child name="Nisha" />;
}
Props flow only one way: Parent ‚Üí Child

4. STATE (Important Concept)

What is State?

State is data that changes over time and controls what appears on the screen.

Examples of state:

  • User logged in or not
  • Counter value
  • Form input text
  • Button enabled / disabled
State is just DATA. It is NOT a function.
Changing state causes the UI to update.

5. useState (Hook / Tool)

What is useState?

useState is a React Hook that allows functional components to create and manage state.

üëâ State = WHAT you store
üëâ useState = HOW you store and update it 
const [count, setCount] = useState(0);

Breaking it down

Part Meaning
count The state value
setCount Function to update state
useState(0) Initial value of state
useState tells React: ‚ÄúPlease remember this value and re-render when it changes.‚Äù

6. State vs useState (CONFUSION CLEARED)

State useState
Data that changes Hook used to create/manage state
Concept Function
Controls UI Triggers UI re-render
Cannot update UI alone Updates state properly
üîë State is the VALUE, useState is the TOOL.

7. Simple Example (Very Important)

function Counter() {
  const [count, setCount] = useState(0);

  return (
    <>
      <p>Count: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        Increase
      </button>
    
  );
}
Clicking the button updates state ‚Üí React re-renders UI
