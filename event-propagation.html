Event Bubbling & Event Capturing
Complete JavaScript Interview & Practical Guide

1. What is an Event?

An event is an action that happens in the browser:

  • click
  • keydown
  • submit
  • mouseover
  • load

When an event occurs, the browser decides how the event travels through DOM elements. This process is called Event Propagation.

When an event occurs, the browser decides in which order elements receive that event.

2. Event Propagation

Event Propagation defines the order in which elements receive an event.

4. Event Bubbling (Default Behavior)

Event Bubbling means the event starts from the target element and moves upward to its parents.

üëâ Bubbling is the default behavior in JavaScript.

Order

child ‚Üí parent ‚Üí grandparent ‚Üí document

Example: Event Bubbling

<script>
document.getElementById("grandparent")
  .addEventListener("click", () => {
    console.log("Grandparent clicked");
  });

document.getElementById("parent")
  .addEventListener("click", () => {
    console.log("Parent clicked");
  });

document.getElementById("child")
  .addEventListener("click", () => {
    console.log("Child clicked");
  });
</script>

Output (Click on Button)

Child clicked
Parent clicked
Grandparent clicked

5. Event Capturing (Trickling Phase)

Event Capturing means the event starts from the top (document) and moves downward to the target.

Capturing does not happen by default.

How to Enable Capturing?

Pass true as the third argument in addEventListener.

Example: Event Capturing

<script>
document.getElementById("grandparent")
  .addEventListener("click", () => {
    console.log("Grandparent clicked");
  }, true);

document.getElementById("parent")
  .addEventListener("click", () => {
    console.log("Parent clicked");
  }, true);

document.getElementById("child")
  .addEventListener("click", () => {
    console.log("Child clicked");
  }, true);
</script>

Output

Grandparent clicked
Parent clicked
Child clicked

8. stopPropagation()

stopPropagation() stops the event from moving further in capturing or bubbling. 

document.getElementById("child")
  .addEventListener("click", (e) => {
    e.stopPropagation();
    console.log("Child clicked only");
  });

Output

Child clicked only

9. Event Delegation (VERY IMPORTANT)

Event Delegation means attaching one event listener to a parent instead of many children.

Event Delegation works because of Event Bubbling.

Example

<ul id="list">
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<script>
document.getElementById("list")
  .addEventListener("click", (e) => {
    if (e.target.tagName === "LI") {
      console.log(e.target.innerText);
    }
  });
</script>
