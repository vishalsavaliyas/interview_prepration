jQuery is a JavaScript library built to make JavaScript:

  • Easier to write
  • Shorter
  • Cross-browser compatible

Simple definition (for fresher):

jQuery helps you write less JavaScript code to do common tasks like DOM manipulation, events, animations, and AJAX. 

Basic rule:
$(selector).action();

  • Complex DOM manipulation

  • Animations were painful

Example

// jQuery
