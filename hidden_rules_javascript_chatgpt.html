1. Truthy & Falsy Values

Only these values are falsy:

false, 0, -0, 0n, "", null, undefined, NaN
Everything else is truthy ‚Äî including [], {}, and "0".

2. == vs ===

0 == false      // true ‚ùå
0 === false     // false ‚úÖ
null == undefined   // true ‚ùå
null === undefined  // false ‚úÖ
Always use ===. == performs type coercion.

3. null vs undefined

  • null ‚Üí intentional empty value (set by developer)
  • undefined ‚Üí value not assigned (JS engine)
typeof null // "object" ‚ùå (JS bug)

4. Hoisting

console.log(a); // undefined
var a = 10;

console.log(b); // ReferenceError
let b = 10;
var is hoisted and initialized. let and const are hoisted but in TDZ.

5. Temporal Dead Zone (TDZ)

{
  console.log(x); // ‚ùå ReferenceError
  let x = 5;
}

6. typeof Weirdness

typeof null        // "object"
typeof []          // "object"
typeof NaN         // "number"
typeof function(){} // "function"
Use Array.isArray() and Number.isNaN().

7. this Keyword Rules

const obj = {
  name: "JS",
  show() {
    console.log(this.name);
  }
};

obj.show(); // JS

const fn = obj.show;
fn(); // undefined
this depends on how the function is called, not where it is written.

8. Arrow Functions

  • No this
  • No arguments
  • Cannot be used with new

9. Object Comparison

{} === {}   // false
[] === []   // false

const a = {};
const b = a;
a === b; // true

10. Primitive vs Reference

let a = 10;
let b = a;
b = 20;
console.log(a); // 10

let obj1 = { x: 1 };
let obj2 = obj1;
obj2.x = 2;
console.log(obj1.x); // 2

11. NaN Rule

NaN === NaN // false ‚ùå

Number.isNaN(NaN) // true ‚úÖ

12. + Operator Coercion

1 + "2" + 3 // "123"
1 + 2 + "3" // "33"

13. setTimeout Is Not Exact

setTimeout(() => console.log("Hi"), 0);
console.log("Bye");

Output:

Bye
Hi

14. Event Loop Priority

console.log("A");

setTimeout(() => console.log("B"), 0);

Promise.resolve().then(() => console.log("C"));

console.log("D");

Output:

A
D
C
B

15. Closures

function counter() {
  let count = 0;
  return function () {
    count++;
    return count;
  };
}

const c = counter();
c(); // 1
c(); // 2

16. for Loop + var Trap

for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 1000);
}
// 3 3 3
Use let instead of var.

17. Default Parameters

let x = 10;
function test(a = x) {
  console.log(a);
}
x = 20;
test(); // 20

18. delete Operator

let a = 10;
delete a; // false

const obj = { x: 1 };
delete obj.x; // true

19. arguments Object

function test() {
  console.log(arguments.map); // undefined
}

20. parseInt Traps

parseInt("10px") // 10
parseInt("px10") // NaN
Always specify radix: parseInt("10", 10)

21. Object.freeze Is Shallow

const obj = Object.freeze({
  a: { b: 1 }
});
obj.a.b = 2; // allowed

22. map / filter Mutation Trap

arr.map(x => x.a = 5); // mutates objects

23. Spread Operator Is Shallow

const a = { x: { y: 1 } };
const b = { ...a };
b.x.y = 2;
console.log(a.x.y); // 2

24. Famous == Interview Trap

[] == ![] // true

25. console.log Is Not Snapshot

const obj = { a: 1 };
console.log(obj);
obj.a = 2;
