JavaScript Polyfill – Complete Interview Guide

What is a Polyfill?

A polyfill is JavaScript code that provides modern functionality to older browsers that do not support it natively.

In simple words: Polyfill imitates a missing feature.

Why Do We Need Polyfills?

  • Older browsers don’t support modern JS features
  • Prevents application crashes
  • Ensures backward compatibility

Example Without Polyfill

const arr = [1, 2, 3];
arr.includes(2); // ❌ Error in old browsers

Example With Polyfill

if (!Array.prototype.includes) {
  Array.prototype.includes = function (value) {
    for (let i = 0; i < this.length; i++) {
      if (this[i] === value) return true;
    }
    return false;
  };
}

Polyfill for Array.map

Array.prototype.myMap = function (callback) {
  const result = [];
  for (let i = 0; i < this.length; i++) {
    result.push(callback(this[i], i, this));
  }
  return result;
};

Polyfill for Array.filter

Array.prototype.myFilter = function (callback) {
  const result = [];
  for (let i = 0; i < this.length; i++) {
    if (callback(this[i], i, this)) {
      result.push(this[i]);
    }
  }
  return result;
};

Polyfill for Array.reduce (Very Important)

Array.prototype.myReduce = function (callback, initialValue) {
  let accumulator = initialValue !== undefined
    ? initialValue
    : this[0];

  let startIndex = initialValue !== undefined ? 0 : 1;

  for (let i = startIndex; i < this.length; i++) {
    accumulator = callback(accumulator, this[i], i, this);
  }
  return accumulator;
};

Polyfill for call()

Function.prototype.myCall = function (context, ...args) {
  context = context || window;
  context.fn = this;
  const result = context.fn(...args);
  delete context.fn;
  return result;
};

Polyfill for apply()

Function.prototype.myApply = function (context, args) {
  context = context || window;
  context.fn = this;
  const result = context.fn(...args);
  delete context.fn;
  return result;
};

Polyfill for bind()

Function.prototype.myBind = function (context, ...args) {
  const fn = this;
  return function (...newArgs) {
    return fn.apply(context, [...args, ...newArgs]);
  };
};

Polyfill vs Transpiler

Polyfill Transpiler
Adds missing features Converts modern syntax
Example: Array.map Example: let → var
Uses core-js Uses Babel

Interview One-Liner

A polyfill is JavaScript code that adds support for modern features in browsers that don’t support them natively.

Important Interview Tips

  • Always check if feature exists before adding polyfill
  • Never override native implementation
  • Common questions: map, reduce, bind, call
