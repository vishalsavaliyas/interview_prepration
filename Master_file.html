1️⃣ JavaScript Core

What is JavaScript?

JavaScript is a single-threaded, synchronous language that handles async tasks using the event loop.

var vs let vs const

  • var – function scoped
  • let – block scoped
  • const – block scoped, no reassignment

Closure


function outer() {
  let count = 0;
  return () => ++count;
}

== vs ===

Always use === to avoid type coercion.

2️⃣ Advanced JavaScript

Event Loop


console.log('A');
setTimeout(()=>console.log('B'),0);
Promise.resolve().then(()=>console.log('C'));
console.log('D');

Output: A D C B

Debounce


function debounce(fn, d) {
 let t;
 return (...a)=>{clearTimeout(t);t=setTimeout(()=>fn(...a),d);}
}

3️⃣ React Fundamentals

What is React?

React is a JavaScript library for building component-based UIs using Virtual DOM.

Props vs State

Props are read-only; state is mutable.

useState


const [count, setCount] = useState(0);

useEffect


useEffect(()=>{ fetchData(); }, []);

4️⃣ Advanced React

Re-render Rules

  • State change
  • Props change
  • Parent re-render

Stale State Fix


setCount(prev => prev + 1);

useMemo vs useCallback


useMemo(()=>calc(a),[a]);
useCallback(()=>fn(a),[a]);
