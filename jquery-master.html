1. What is jQuery?

jQuery is a fast, small, and feature-rich JavaScript library. It simplifies:

  • DOM manipulation
  • Event handling
  • Animations
  • AJAX

Rule:

$(selector).action();

2. jQuery Ready Function

Ensures HTML loads before JavaScript runs.

$(document).ready(function() {
    // code here
});

Short version:

$(function() {
    // code here
});
