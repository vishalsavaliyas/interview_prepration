AJAX – Zero to Hero (Interview + Practical Master File)

1. What is AJAX?

AJAX stands for Asynchronous JavaScript And XML.

AJAX is a technique that allows a web page to communicate with a server without reloading the page.

Example: Like button, live search, chat messages.

2. Why AJAX is Needed?

Before AJAX

  • Page reloads on every request
  • Slow user experience
  • More bandwidth usage

After AJAX

  • No page reload
  • Fast & smooth UI
  • Modern web applications

3. How AJAX Works (Flow)

User Action (Click / Load)
        ↓
JavaScript sends request
        ↓
Server processes request
        ↓
Server sends response (JSON)
        ↓
JavaScript updates UI

4. Is AJAX a Language?

No. AJAX is a combination of:

  • JavaScript
  • Browser APIs
  • HTTP
  • Server (Backend API)
  • JSON / XML

5. Data Format Used

JSON (Most Common)

{
  "name": "Amit",
  "age": 25
}

6. Ways to Use AJAX

  • XMLHttpRequest (Old)
  • Fetch API (Modern & Recommended)
  • Axios (Library – React friendly)

7. AJAX using Fetch API (GET Request)

fetch(url)
  .then(response => response.json())
  .then(data => console.log(data))

8. AJAX using Async / Await

async function loadData() {
  const response = await fetch(url);
  const data = await response.json();
}

9. AJAX POST Request (Send Data)

fetch(url, {
  method: "POST",
  headers: {
    "Content-Type": "application/json"
  },
  body: JSON.stringify({
    title: "Hello",
    body: "AJAX Example"
  })
})

12. AJAX in React

useEffect(() => {
  fetch("/api/users")
    .then(res => res.json())

13. Common Interview Questions

Q: What is AJAX?

A: Technique to communicate with server asynchronously without page reload.

Q: Is AJAX synchronous?

A: No, it is asynchronous.

Q: Does AJAX use HTTP?

A: Yes.

Q: Is Fetch AJAX?

A: Yes, Fetch is modern AJAX.
