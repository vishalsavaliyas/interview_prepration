1. What is JavaScript?

JavaScript is a single-threaded, synchronous language that supports asynchronous programming using the event loop.

2. var vs let vs const

  • var - function scoped
  • let - block scoped
  • const - block scoped, no reassignment

3. Closure

  let count = 0;
  return function() {
    count++;
    return count;
  }
}
