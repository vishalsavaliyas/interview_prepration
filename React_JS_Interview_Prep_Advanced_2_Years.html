Advanced JavaScript Interview Questions

1. Explain the Event Loop in JavaScript

The event loop allows JavaScript to handle asynchronous operations by executing synchronous code first, then microtasks (Promises), then macrotasks (setTimeout).

2. What are Promises and async/await?

Promises represent future values. async/await is syntactic sugar over promises for cleaner asynchronous code.


async function fetchData() {
  const res = await fetch(url);
  const data = await res.json();
  return data;
}

3. Shallow vs Deep Copy


const shallow = {...obj};
const deep = JSON.parse(JSON.stringify(obj));

4. Debouncing vs Throttling

  • Debounce: Executes after delay ends
  • Throttle: Executes at fixed intervals

5. Call, Apply, Bind


function greet(city) {
  console.log(this.name + " from " + city);
}
greet.call({name:"A"}, "Delhi");

Advanced React Interview Questions

1. What is Virtual DOM?

Virtual DOM is a lightweight copy of real DOM that React uses to optimize UI updates.

2. What is reconciliation?

Reconciliation is React’s process of comparing old and new Virtual DOM trees to update only changed nodes.

3. useMemo vs useCallback


const memoValue = useMemo(() => heavyCalc(a), [a]);
const memoFn = useCallback(() => doSomething(a), [a]);

4. React.memo

Prevents unnecessary re-renders of functional components.

5. Controlled vs Uncontrolled Components

Controlled components use state; uncontrolled use refs.

Task 4: Debounce Input


function debounce(fn, delay) {
  let timer;
  return function(...args) {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), delay);
  }
}
